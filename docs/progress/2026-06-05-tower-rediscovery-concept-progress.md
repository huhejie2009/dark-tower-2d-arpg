# 2026-06-05 通天塔重新发现概念图记录

## 目标调整

本次方向从“神罚后的荒败现场”调整为“几千年后的人们重新发现远古通天塔”。

核心叙事：

- 通天塔由远古人类建造，用于抵达天界。
- 塔在远古时代之后被封存、遗忘。
- 几千年后，后来的人们在平原上重新发现它。
- 塔内已经成为怪物盘踞之地，但概念图不表现战斗，而表现发现瞬间的沉默、敬畏和冷峻。

## 当前美术判断

这一版比前一版更接近目标：

- 光效明显减少。
- 塔体更像冷灰水泥/混凝土巨构。
- 华丽仙宫感降低。
- 前景探索者强化了“几千年后重新发现”的尺度和叙事。
- 小河和低平地貌保留了项目世界观需要的平原环境。

仍需后续控制：

- 继续压低装饰性古风屋檐比例。
- 避免塔过度像奇幻大教堂。
- 进一步增加粗野主义、苏式纪念碑和人造工程感。
- 塔体入口可逐步设计为“封存/禁入/危险”的明确符号。
