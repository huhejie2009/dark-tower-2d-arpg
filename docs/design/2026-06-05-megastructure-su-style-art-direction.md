# 2026-06-05 巨构苏式冷塔美术方向

## 方向修正

此前战斗背景参考了《梦幻西游》式等距室内场景，但这不是最终世界观方向。新的统一方向应围绕：

- 一座直冲云霄的 Megastructure 巨塔。
- 巨塔坐落在极大的草原或平原上。
- 平原中有小河流经过，形成进入巨塔前的地理记忆点。
- 建筑色彩和局部装饰参考苏式建筑语言：白墙、黑蓝瓦、低饱和朱红木构、漏窗、月洞门、层叠屋檐。
- 整体气质是冰冷、孤高、危险、静默，而不是热闹城镇或传统仙侠街区。
- 概念过场图和过场动画可以使用更强的漫画幻想色彩：高饱和冷暖对比、通透光感、极端尺度、云雾和风。

## 美术管线分层

### 1. 世界观/概念过场图

用于：

- 开场图。
- 楼层章节过场。
- 主菜单背景。
- 宣传图或世界观设定图。

风格允许更绘画化、更戏剧化：

- 巨塔极高，顶部进入云层甚至消失在天空中。
- 草原、河流、天空和塔形成强构图。
- 色彩可以偏漫画幻想，但保持原创，不直接模仿特定艺术家或 IP。
- 角色可以非常小，用于强调巨构尺度。

### 2. 游戏内战斗场景

用于：

- 实际爬塔楼层。
- 战斗房间。
- UI 背景或楼层节点图。

风格需要更可读：

- 不追求过场图的复杂光影。
- 地面、障碍、敌人、掉落、传送门必须清晰可辨。
- 塔内场景应继承巨塔设定：冷白石、黑蓝瓦影、朱红木构、青灰金属、冰冷灯光。
- 场景可以有苏式元素，但不能做成热闹古镇。

### 3. 角色与怪物素材

用于：

- 玩家职业 SpriteSheet。
- 普通敌人、精英、Boss。
- 技能特效和掉落。

方向：

- 角色服饰可以有中式幻想元素，但更偏“冷塔探索者/守塔者/异化者”。
- 敌人应从巨塔生态中生长出来：石像守卫、冷铁机关、霜蚀修士、塔影异物。
- 避免过度可爱、热闹、Q 版化。

## 建议废弃或降级的旧方向

- 不再把《梦幻西游》式室内热闹场景作为世界观主方向。
- 已接入的等距室内背景可暂时作为测试背景保留，但后续应替换为“冷塔内部”环境。
- 当前 IMAGE2 玩家单帧可继续作为临时美化素材，但后续角色设计要转向更冷、更塔内探索者的统一风格。
