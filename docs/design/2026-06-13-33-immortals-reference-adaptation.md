# 33 Immortals 参考拆解与本项目适配方案

日期：2026-06-13

## 目的

本文件用于记录《33 Immortals》对《Dark Tower 2D ARPG》的可借鉴方向。目标不是照搬它的 33 人合作 Roguelike 结构，而是抽取其中适合本项目的战斗节奏、视觉可读性和神罚叙事压力，并转化为单机 2D 暗黑刷宝/爬塔 ARPG 的设计语言。

公开资料显示，《33 Immortals》核心定位是 33 人合作动作 Roguelike，强调快速进入战斗、怪潮、Boss、合作能力、表情/信号协作、神怒压力和每轮变强。我们只借鉴其中适合单机爬塔 ARPG 的部分。

## 当前明确不借鉴的部分

### 不做局内遗物/短局成长系统

暂时不引入“每局选择遗物、每局构筑变化、局内成长树”这类 Roguelike 局内成长。原因：

- 当前项目的核心仍是暗黑刷宝/装备成长/爬塔推进。
- 背包、装备、主城、战斗可读性和素材管线尚未完全稳定，过早加入局内成长会分散系统重心。
- 后续如果需要短局构筑，可以作为 P4 以后独立系统重新评估，而不是混进当前 P2/P3。

## 可借鉴方向 A：战斗房间节奏

《33 Immortals》的战斗不是单纯走廊刷怪，而是由探索、目标房、怪潮、奖励、Boss 压力组合成连续节奏。本项目可将塔层改造成更明确的节点结构：

- 入口缓冲区：玩家进入楼层后有短暂安全区域。
- 战斗房：清怪、守点、封印、击杀精英等明确目标。
- 奖励房：宝箱、商人、修整、传送选择。
- Boss 门：满足房间目标后开启。

### 对本项目的实现边界

- 先做单层内的房间目标，不做复杂随机地图。
- 每个房间只需要一个清晰目标。
- 房间奖励仍然走现有掉落、金币、装备数据，不新增局内遗物。

### 验收标准

- 玩家能在 3 秒内理解当前房间目标。
- 完成目标后有明确反馈：门开、奖励出现、HUD 提示。
- 单层至少包含 2 种不同节奏：普通战斗和精英/事件战斗。

## 可借鉴方向 B：神罚压力机制

《33 Immortals》使用“God's Wrath”强化叙事压力。本项目世界观中，通天塔本身就是被神明惩罚后的禁忌建筑，因此可以引入“神明凝视/塔之审判”作为房间压力机制。

可用触发条件：

- 玩家在房间停留过久。
- 玩家开启高价值宝箱。
- 玩家击杀精英怪后。
- 玩家进入 Boss 门前。

可用表现：

- 地面出现冷色暗光裂痕。
- 危险区域先显示预警圈，再落下惩罚攻击。
- 屏幕边缘轻微暗化，但不遮挡 UI。
- 追加怪物增援或短时间封锁出口。

### 对本项目的实现边界

- 只做战斗压力，不做随机惩罚表。
- 必须先有清楚预警，再造成伤害。
- 不能用大面积特效遮挡敌人攻击前摇。

### 验收标准

- 神罚伤害前至少有 0.6 秒可见预警。
- 玩家能通过走位躲避，不是纯数值惩罚。
- 同屏危险区域、玩家、敌人和掉落物不互相遮挡。

## 可借鉴方向 C：协作技能的单机化改造

《33 Immortals》的合作能力强调多人合力释放。我们不做 33 人联机，但可以把这种“合力释放”的视觉和节奏转化为单机系统：

- 灵魂回响：玩家积累能量后召唤短暂影子协助攻击。
- 契约技：装备或职业技能触发一次大范围支援。
- 战场信标：玩家放置短暂标记，延迟爆发或吸引敌人。

### 对本项目的实现边界

- 不做联网。
- 不做多人同步。
- 不做局内遗物绑定。
- 先作为技能/VFX 管线接口，为职业技能和装备词条服务。

### 验收标准

- 技能前摇、释放、命中、消散四段清楚。
- 技能特效和人物攻击动画分层，不烘焙在同一张角色帧里。
- 技能必须能被 HUD 或冷却提示解释。

## 可借鉴方向 D：俯视角战斗可读性

《33 Immortals》值得借鉴的核心之一是俯视角下大量单位、技能和危险区仍保持清楚。本项目当前最需要补强的也是这一点。

本项目应建立以下战斗视觉规则：

- 玩家层级永远高于地面提示，低于 UI。
- 敌人攻击前摇必须有明显动作，不允许只用抖动代替。
- 敌人危险区、玩家技能区、掉落物提示使用不同颜色族。
- 打击特效与人物/怪物动作分开制作。
- 武器与身体分层，方便后续换武器。

### 验收标准

- 玩家能在 1 秒内分辨敌人是否正在攻击。
- 玩家能分辨自己技能、敌人技能、掉落物、交互物。
- 战斗截图缩小到 50% 后仍能看清玩家位置和主要威胁。

## ROADMAP 调整建议

### P2：战斗可读性与房间目标

- 增加房间目标系统雏形。
- 增加至少 2 种房间节奏。
- 增加敌人攻击预警和动作状态 QA。
- 建立神罚压力的最小可玩版本。

### P3：塔层节奏与技能表现

- 增加精英房、奖励房、Boss 门。
- 增加 1 个灵魂回响/契约技原型。
- 增加神罚预警、伤害、冷却和视觉分层规则。

### 暂不进入 ROADMAP

- 局内遗物选择。
- Roguelike 每局构筑。
- 多人协作。
- 网络同步。

## 合作注意

与 co-worker 协作时，本参考方案只作为系统和视觉可读性方向，不直接改动当前美术主风格。当前美术方向仍保持“冷峻巨构通天塔、草原/平原、暗光核心、无山、冷色压迫感”，角色素材继续走“高精像素角色 + 分层武器 + 独立特效”的资产管线。
